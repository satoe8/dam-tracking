# Dam Tracker Mobile App

A React Native habit-tracking game with animated beaver pets!

## Setup

1. Install dependencies:
```bash
cd mobile
npm install
```

2. Create `.env` file:
```env
EXPO_PUBLIC_SUPABASE_URL=your_supabase_url
EXPO_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

3. Start the app:
```bash
npm start
```

4. Run on device:
- iOS: Press `i` or scan QR code with Camera app
- Android: Press `a` or scan QR code with Expo Go app

## Dev Mode

The app includes a dev mode with local storage for testing without Supabase.

## Features

- Habit tracking with daily/weekly frequencies
- Animated beaver pet with energy levels
- Friends system
- Calendar view of completed habits
- Confetti animations
- Touch-friendly mobile UI
